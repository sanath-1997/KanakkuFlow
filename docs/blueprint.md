# **App Name**: Kanakku Flow

## Core Features:

- Trilingual Onboarding: Localized interface welcoming users in English, Tamil, and Malayalam with an initial language selector for immediate accessibility.
- Live Ledger Balance: Real-time calculation and prominent display of the current balance at the top of the interface.
- Dual-Action Transaction Logging: High-visibility controls with a green income button and a red expense button for rapid data entry.
- Generative Category Mapping: An AI tool that intelligently suggests categories and emoticons based on the transaction description text provided by the user.
- Interactive Spending Pie Chart: A dynamic visualization tool located below the primary buttons that breaks down expenses by category.
- Custom Category Studio: Management interface to add or remove categories, including a picker for emoticons to represent each expense type.
- Time-Stamped History: A detailed feed of all recent transactions with precise date and time formatting for auditability.

## Style Guidelines:

- Primary color is a sophisticated Royal Blue (#2855BF), evoking trust and financial clarity.
- The background is a very soft, desaturated lavender-tinted white (#F8F9FD) for a high-end agency look.
- Functional accents include a vibrant Emerald Green (#12B76A) for income and a bright Ruby Red (#F04438) for expenses.
- Headlines and numeric balance figures use 'Space Grotesk' for its tech-forward, high-contrast feel. Body text uses 'Inter' for optimal readability.
- Minimalist glyphs paired with expressive colorful emojis to distinguish between financial categories like groceries, utilities, and transport.
- A focused single-column dashboard centered around vertical rhythm, with significant whitespace separating the balance card from the interactive charts.
- Fluid, high-frequency motion transitions (Spring physics) when toggling between income and expense categories or switching languages.
